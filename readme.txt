In order to use the program you need:

On windows: enter "mingw32-make" (by default) or "make" into the console, after that the program 
will compile and you will need to manually open the file output.html (by default).

On linux: enter "make linux" in the console, after that the program will compile and open the file output.html 
(by default) in the browser.

To change the parameters in the "Makefile":

main - the name of the file in .c format

matches - the name of the file with matches

players - the name of the file with the players

output - the name of the file to output information in the format .html
